注册

1、进入蓝图百货首页，点击左上角注册或是右上角免费注册。

2、进入创建账户页面，填写完信息后，勾选我已阅读并同意 《蓝图百货用户协议》，点击下一步

3、进入创建账户页面，设置本人信息，也可以直接跳过此步。

3、恭喜你成功完成了注册，点击立即体验快去体验蓝图百货带给你的惊喜吧。

{% elif help == 'dl' %}

登录

1、点击登录字样跳转到登录页面，输入注册时设置的用户名和密码登录

{% elif help == 'nh_qy' %}

农行签约

1、第一步，设置资金密码（执行农行签约需先设置此项），以便支付时使用

2、第二步，执行签约流程，*为必填项，其他为选填项，信息填好后点击下一步

3、签约成功，开启愉快的购物之旅

3、签约失败，返回查看，重复第二步的操作

{% elif help == 'wx_zf' %}

微信支付

1、选择微信支付，跳转到微信支付页面，微信支付页面会出现微信支付的二维码，用户需打开自己的微信，点击发现找到扫一扫，扫描二维码即可进行微信支付，二维码有时间限制，请尽快支付，如果二维码过期，可点击刷新按钮重新获取二维码，点击页面内的订单详情，可以看到购买的订单信息。

{% elif help == 'sh_dz' %}

收货地址

1、我的账户-提货点页面，如果您选择的是货运，在此页面可以设置自己的收货地址，经常用到的收货地址可以设置为默认收货地址，点击新建收货地址可以添加更多的收货地址。

2、点击新建收货地址,进入新页面，在此页面填写完收货人信息后点击保存收货地址即可。

{% elif help == 'th_dz' %}

提货点

1、我的账户-提货点页面，如果您选择的是自提，在此页面可以选择距离自己较近，提货比较方便的自提点，经常用到的自提点可以设置为默认自提点，点击新建自提地址可以选择更多的自提点。

2、点击新建自提地址后，进入新页面，选择自己所在区域检索到所在区域内的自提点，然后选择自己想要的自提点，点击保存自提点即可。经常用到的自提点也可以设置为默认自提点。

{% elif help == 'jy_jl' %}

交易记录

1、我的账户-交易记录页面，在此页面可以查询自己的交易情况，可以按照关键字，时间，交易方式进搜索，可以按照商品名称和成交时间进行排序。每页最多有20条记录。
